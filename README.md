# CIFAR-10

CIFAR是一个影响力很大的图像分类数据集，分为了CIFAR-10 和CIFAR-100 两个问题，其中的图片是由Alex Krizhevsky, Vinod Nair和Geoffrey Hinton收集的。

CIFAR-10包含了来自10个不同种类，总共60,000张32x32的彩色图片，每一类含有6000张图片。进一步地，其中50,000张为训练图片，剩余的10,000张为测试图片。
